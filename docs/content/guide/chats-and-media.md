# Chats and Media

Use this guide for text messages, image sharing, album sharing, and local history behavior.

## Send text messages

- Open a conversation.
- Type your message.
- Press Send.

[picture: message composer with text and send button]

## Send image messages

- Use Attach image in the chat composer.
- Pick an image file.
- Wait for upload progress to complete.

[picture: image upload progress bar in chat composer]

## Share albums

- Click Share album.
- Pick an album from the picker.
- Confirm share to the current conversation.

[picture: album picker panel with Share buttons]

## Open album media

- In a message that contains an album, click Open.
- Click outside the album viewer container to close.

[picture: album viewer overlay with media grid]

## Local history behavior

Open Grind stores local chat history for resilience.

- Local-only entries are labeled as from local history.
- Local history can be cleared per conversation from the chat header.

[picture: chat header Clear local history button]

## Failed sends

For messages that failed to send:

- Retry text messages from the failed item.
- Delete/unsend local failed entries directly from the message menu.

[picture: failed message row with retry and actions]
