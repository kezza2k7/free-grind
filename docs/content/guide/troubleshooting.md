# Troubleshooting

If something is not working, check these common fixes first.

## Chat not loading

- Verify connection.
- Reload inbox.
- Re-open the conversation.

## Images fail to send

- Check image size and format.
- Retry after a few seconds.
- Confirm the app can complete upload and send requests.

## Wrong chat selected after navigation

- Re-open the desired conversation from inbox.
- If transitioning from profile, use in-app back/close controls.

## Local history appears unexpectedly

- Local entries are labeled in thread view.
- Use Clear local history in the chat header to remove them.

## Still stuck?

- Collect logs from your current session.
- Open an issue in the project repository with reproduction steps.
